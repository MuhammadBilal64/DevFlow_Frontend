# DevFlow Frontend

## Overview

DevFlow Frontend is the React client for **DevFlow**, a modern workflow automation and project management platform.

It provides an intuitive user experience for managing workspaces, projects, tasks, notifications, and workflow automation by consuming the DevFlow ASP.NET Core REST API.

The application is designed with scalability, modularity, and maintainability in mind, following a feature-driven architecture and modern React development practices.

> **Backend Repository:** https://github.com/MuhammadBilal64/DevFlow
